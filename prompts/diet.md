# 饮食记录提示词

## 使用说明
用户发送食物图片、说"吃了什么"、"记录饮食"、"这顿多少热量"时使用。

---

## 提示词

```
你正在执行饮食记录。

用户描述了饮食或发送了食物图片，请解析并记录。

### 解析规则
1. 识别食物种类和份量
2. 查阅 knowledge-ref/健身饮食参考资料.md 中的食物营养率数据
3. 估算热量、蛋白质、碳水
4. 所有估算值用 ~ 前缀（如 ~600大卡）
5. 判断当天是休息日还是力训日，对比配额

### 每日配额
- 休息日：1628大卡 / 碳水174g / 蛋白质98g / 脂肪60g
- 力训日：1723大卡 / 碳水189g / 蛋白质103g / 脂肪60g
- 力训日判断：当天安排了力训（背/胸/腿）则为力训日

### 输出步骤
1. 列出食物明细表（食物 | 份量 | 热量 | 蛋白质 | 碳水）
2. 汇总本餐数据
3. 汇总今日已摄入 + 剩余配额
4. 如果蛋白质不足，建议高蛋白补充方案
5. 如果热量超标，提醒并建议后续调整
6. 写入 health/diet.md 记录表
7. 更新当日汇总

### 写入格式
在 health/diet.md 的记录表中添加一行：
| YYYY-MM-DD | 早餐/午餐/晚餐 | 食物描述 | ~XXX大卡 | ~XXg | ~XXg |

### 常见食物速查
- 鸡胸肉：165大卡/100g，蛋白质23%
- 鸡蛋：155大卡/100g，蛋白质13%
- 米饭(一般)：30%碳水率
- 包菜/西兰花：~34大卡/100g
- 需避免：高脂肉、糖油混合物、吸油菜
- 详细数据见 knowledge-ref/健身饮食参考资料.md
```
