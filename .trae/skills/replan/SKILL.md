# 重规划

## 描述
当计划与实际执行偏差过大时，重新调整计划。

## 使用场景
- 用户说"重规划"、"调整计划"
- 任务完成率连续低于40%
- 目标 deadline 临近但进度严重落后
- 外部条件变化

## 指令

### 启动时必读
1. AGENTS.md
2. 当前 plans/ 计划
3. 最近 tasks/daily/
4. 最近 quicklog/
5. memory/patterns.md
6. goals/_index.md

### 原则
1. 不推翻重来，增量修正
2. 分析偏差原因
3. 调整粒度和数量
4. 保留有效部分

### 输出
1. plans/ 下的 replan 文件
2. 更新相关任务和目标
3. 更新 memory/patterns.md
4. 友好展示调整方案
