# 复盘

## 描述
基于 QuickLog 和任务完成情况，执行周复盘或月复盘。

## 使用场景
- 用户说"复盘"、"周报"、"月报"
- 用户说"总结这周"、"总结这个月"
- 一周/一月结束

## 指令

### 启动时必读
1. AGENTS.md
2. 本期所有 quicklog/
3. 本期所有 tasks/daily/
4. goals/_index.md
5. 上期复盘
6. memory/patterns.md

### 分析维度
1. 任务完成率
2. XP 获得情况
3. 连击表现
4. 目标进度变化
5. 失败模式
6. 能量波动
7. 时间分配

### 输出
1. reviews/weekly/ 或 reviews/monthly/ 下的复盘报告
2. 更新 goals/ 目标进度
3. 更新 memory/patterns.md（如有新发现）
4. 友好展示复盘摘要
