# 饮食记录

## 描述
Stone Plan 的饮食记录技能。支持文字描述和图片识别，自动估算热量、蛋白质、碳水，对比每日配额。

## 使用场景
- 用户说"吃了XX"、"记录饮食"、"这顿多少热量"
- 用户发送食物图片
- 用户说"估一下热量"、"今天吃了多少"
- 每日三餐记录（早餐/午餐/晚餐）

## 指令

### 启动时必读
1. AGENTS.md
2. health/diet.md — 饮食记录和每日配额
3. knowledge-ref/健身饮食参考资料.md — 食物营养率数据
4. goals/active/goal-fitness.md — 减脂目标配额

### 解析规则
1. **文字输入**：提取食物名称、份量，查营养率表估算
2. **图片输入**：识别食物种类和份量，估算热量
3. **估算标注**：所有估算值用 `~` 前缀表示（如 ~600大卡）
4. **配额对比**：对比每日配额（休息日/力训日），显示剩余

### 每日配额（来自 goal-fitness）
- 休息日：1628大卡 / 碳水174g / 蛋白质98g / 脂肪60g
- 力训日：1723大卡 / 碳水189g / 蛋白质103g / 脂肪60g

### 输出步骤
1. 估算热量/蛋白质/碳水
2. 写入 health/diet.md 的记录表
3. 更新当日汇总（已摄入/剩余配额）
4. 如果蛋白质不足，建议补充高蛋白食物
5. 如果热量超标，提醒并建议调整

### 饮食记录格式
| 日期 | 餐次 | 内容 | 估算热量 | 蛋白质 | 碳水 |
|------|------|------|---------|--------|------|
| YYYY-MM-DD | 早餐/午餐/晚餐 | 食物描述 | ~XXX大卡 | ~XXg | ~XXg |

### 常见食物速查（详见 knowledge-ref/健身饮食参考资料.md）
- 鸡胸肉：165大卡/100g，蛋白质23%
- 鸡蛋：155大卡/100g，蛋白质13%
- 米饭(一般)：30%碳水率，外食标准盒约100-120g碳水
- 需避免：高脂肉、糖油混合物、吸油菜
