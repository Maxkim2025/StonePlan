# 计划生成

## 描述
基于当前目标树和任务池，生成周计划或月计划。

## 使用场景
- 用户说"周计划"、"月计划"、"生成计划"
- 用户说"安排这周"、"安排这个月"
- 新的一周/月开始

## 指令

### 启动时必读
1. AGENTS.md
2. goals/_index.md
3. tasks/backlog/
4. memory/patterns.md
5. 最近 reviews/

### 规则
1. 优先级排序
2. 每日任务数量由目标紧迫度驱动，不设固定上限（正常4-8个，考前冲刺可10+）
3. 高低耗能搭配
4. 预留休息日
5. 考虑失败记忆

### 输出
1. plans/weekly/ 或 plans/monthly/ 下的计划文件
2. tasks/daily/ 每日任务文件
3. 友好展示计划摘要
