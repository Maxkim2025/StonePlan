# 目标拆解

## 描述
将用户的目标拆解为可执行的子目标和每日任务。

## 使用场景
- 用户说"我想学吉他"、"我要减到140斤"
- 用户说"新目标"、"创建目标"、"拆解目标"
- 用户描述一个想要达成的结果

## 指令

### 启动时必读
1. AGENTS.md
2. goals/_index.md（避免重复）
3. memory/patterns.md

### 拆解原则
1. 语义理解优先
2. 最小可感知
3. 有明确边界
4. 时间可控（<30分钟/任务）
5. 可量化

### 流程
1. 理解目标含义和约束
2. 确定时间（estimated_duration + deadline）
3. 拆解子目标（如有必要）
4. 拆解每日任务
5. 分配 XP
6. 检查失败记忆

### 输出
1. goals/active/goal-{{slug}}.md
2. 更新 goals/_index.md
3. 生成首批 tasks/daily/ 文件
4. 更新 tasks/_index.md
